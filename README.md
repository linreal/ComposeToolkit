# ComposeToolkit

A collection of Kotlin compiler plugins and tools for Jetpack Compose performance analysis and optimization.

## Features

- Jetpack Compose recomposition tracking
- Performance monitoring tools
- Kotlin multiplatform support
- Custom compiler plugins for Compose optimization

## Project Structure

See [STRUCTURE.md](STRUCTURE.md) for detailed project organization.

## Getting Started

This project uses Gradle for build management. To get started:

```bash
./gradlew build
```
